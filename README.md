# pyNeo3DLib

3D 데이터 처리를 위한 파이썬 라이브러리입니다.

## 설치 방법

### pip를 사용한 설치

```bash
pip install pyneo3dlib
```

### 개발 버전 설치

최신 개발 버전을 설치하려면 GitHub 저장소에서 직접 설치할 수 있습니다:

```bash
pip install git+https://github.com/사용자명/pyNeo3DLib.git
```

### 소스에서 설치

```bash
git clone https://github.com/사용자명/pyNeo3DLib.git
cd pyNeo3DLib
pip install -e .
```

## 요구사항

- Python 3.10 이상
- numpy
- 기타 필요한 의존성 패키지
